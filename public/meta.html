🌎🌏🌍Allow same origin resources using meta element 🌎🌏🌍

Options in url params

  • No QueryString -> Tries to load script from the same domain
  • script=allow_other_origin -> Tries to load script from different domain example: http://dy.omer.com:3000/?policy=allow_other_origin&script=https://omerherera.github.io/cdn/src/script.js
  • policy=block_eval -> Allow script execution from certain domains while blocking the use of eval(
  • policy=allow_same_origin -> Allow script execution from only from the same domain
  • policy=allow_same_origin_blocking_eval -> Allow script execution from same domain while blocking the use of eval()
  • policy=allow_other_origin&script=https://omerherera.github.io/cdn/src/script.js -> Allow script execution from different domain + the use of eval()
